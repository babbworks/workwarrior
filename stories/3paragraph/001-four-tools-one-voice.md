# Four Tools, One Voice

TaskWarrior is one of the most powerful task management tools ever built for the terminal. It has urgency scoring, dependencies, project hierarchies, custom attributes, and a query language that makes most databases jealous. TimeWarrior tracks exactly where your hours go with a precision that humbles any manual timesheet. JRNL gives you a searchable, encrypted, plaintext journal that you actually own. Hledger is double-entry accounting for people who want to understand their money without handing it to a subscription app. Each of these tools is exceptional. Each was also built by a different team, with a different philosophy, storing data in a different place, configured through a different file.

Using them together has always required friction. You configure TaskWarrior for work. You configure it again for personal use. TimeWarrior doesn't know which configuration is active. JRNL is just always there, logging to the same place regardless of context. You end up with workarounds — a shell script here, a wrapper there, a convention you have to remember to follow. The tooling works. The integration is yours to figure out.

Workwarrior figures it out for you. It wraps all four tools under a single `ww` CLI and routes every command to the right binary, the right data directory, the right config — automatically, based on your active profile. It doesn't replace the tools. It doesn't hide them. When you need to go deeper than `ww` allows, the tools are right there, already pointed at the correct profile data. Workwarrior is the voice that speaks to all four at once, and ensures they all hear the same name.
