# Workwarrior Stories

Creative writing about what Workwarrior is, how it works, and why it matters.
Each piece is standalone. Length tiers are for practical reuse — elevator pitches,
website copy, blog posts, README intros, talks.

## Structure

| Folder | Use |
|---|---|
| `statements/` | One-liners and punchy declarations — taglines, slide decks, social posts |
| `1paragraph/` | Single-scene vignettes — README leads, feature intros, email openers |
| `2paragraph/` | Problem + solution — landing page sections, pitch deck slides |
| `3paragraph/` | Deeper story — blog post section, documentation intro |
| `4paragraph/` | Full argument — medium post, product brief |
| `5paragraph/` | Narrative arc — case study, feature story |
| `6ormoreparagraphs/` | Long-form — deep dive articles, white papers, talks |
