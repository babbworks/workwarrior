# Noctalia Custom Plugins

Custom plugins for Noctalia Shell by hnsstrk.

## Installation

1. Open Noctalia Settings → Plugins → Sources
2. Click "+ Add custom repository"
3. Enter URL: `https://github.com/hnsstrk/noctalia-custom-plugins`
4. Plugins will appear in the "Available" tab

## Plugins

| Plugin | Version | Description |
|--------|---------|-------------|
| Taskwarrior | 2.0.2 | A full-featured Taskwarrior client |
